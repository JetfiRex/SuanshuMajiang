# SuanshuMajiang
算数麻将番数计算程序

本计算程序打算专门为了算数麻将计算番数。我们将用Python语言去计算番数。目前打算使用Python先计算给定胡法的，然后在考虑给定牌型搜索胡法或者确定是否可以胡。
